# 🌤️ React Global Weather App

A fast, interactive weather application built with React. This app allows users to search for real-time weather conditions in any city worldwide, featuring a clean, professional aesthetic and dynamic UI elements.

## ✨ Features

* **Live Conditions:** Displays current temperature, humidity, and wind speed.
* **7-Day Forecast:** Provides a detailed, day-by-day weather outlook.
* **Dynamic UI:** Features animated weather icons that update based on live conditions.

## 🛠️ Tech Stack

* **Framework:** React.js
* **Styling:** CSS3 (Custom responsive design)
* **API Integration:** SheCodes Weather API
